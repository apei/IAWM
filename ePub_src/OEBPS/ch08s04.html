8.4. Detección de dispositivos

Los sistemas de detección de dispositivos móviles reconocen el dispositivo o el navegador desde el que se accede a un sitio web y redireccionan a la versión adaptada correspondiente. Se trata de una forma eficaz de que los lectores puedan llegar a la versión para móviles automáticamente, sin tener que buscar enlaces, y desde la misma URL que utilizan desde un terminal fijo. Para ello basta con introducir unas líneas al código fuente de nuestro sitio web[8]. Algunos de los servicios en línea de alojamiento y creación de sitios web para móviles generan automáticamente el código fuente para la detección de dispositivos a introducir en nuestro sitio web.


[8] En el Anexo 1 se indican direcciones en las que se explican métodos para la detección de dispositivo.
